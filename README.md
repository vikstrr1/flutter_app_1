# Finance Tracker

## Description
A responsive personal finance tracker designed to help users manage their budget effectively. The application allows users to log expenses and income, automatically calculating the current balance. It provides a clear dashboard to analyze spending habits broken down by category.

## Deployment
**Live App:** <https://finance-tracker-ac8ac.web.app/>

## Key Features
* **Balance Tracking:** Real-time calculation of total income, expenses, and remaining balance.
* **Categorization:** Organize transactions into categories (e.g., Food, Transport, Bills) for better insights.
* **Transaction Management:** Easily add, edit, or delete transactions.
* **Responsive Design:** Works seamlessly on mobile and desktop/tablet views.

## Usage
1. **Dashboard Overview:** View your current balance, total spent, and income summary immediately upon opening the app.
2. **Add Transactions:**
   * Click **"Add New Expense"** to log spending.
   * Click **"Add New Income"** to log earnings (e.g., Salary).
3. **Analyze Spending:** Check the "Spending by Category" card to see a breakdown of where your money is going.
4. **Manage History:**
   * Click **"View All Transactions"** to see your full history.
   * **Edit:** Tap on any item in the "Recent Transactions" list or the "All Transactions" screen to modify details.
   * **Delete:** Use the trash icon to remove an entry.
